# specs/

Acá se guarda una carpeta por cada feature, generada por el ciclo de trabajo.

Cada feature tiene su propia carpeta numerada con tres archivos:

```
specs/
└── 001-nombre-feature/
    ├── spec.md      → QUÉ construir y por qué   (lo genera /especificar)
    ├── plan.md      → CÓMO construirlo          (lo genera /planificar)
    └── tasks.md     → lista de tareas           (lo genera /tareas)
```

Empezá una feature nueva con el comando `/especificar`.
